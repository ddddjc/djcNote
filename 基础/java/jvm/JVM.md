## 什么是JVM

定义：Java Virtual Machine -java程序的运行环境 （Java 二进制字节码运行环境）

好处：

- 一次编写，到处运行
- 自动内存管理，垃圾回收功能
- 数组下标越界检查
- 多态

比较：
